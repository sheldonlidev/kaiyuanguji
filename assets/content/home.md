> 通过技术手段推动古籍的数字化、校对及开源存储，构建古籍知识图谱与 AI 模型

---

## 项目愿景

本项目致力于利用现代技术解决古籍数字化过程中的核心难题，从**专业排版**到**高效校对**，从**智能识别**到**开放共享**，最终实现古籍的**智能化研究**。

我们相信，通过开源协作的力量，可以让中华文化遗产的数字化变得更加高效、准确、可持续。

---

## 🚀 五步走发展路线图

### 第一阶段：中文古籍排版支持 ✨

**核心目标**：实现符合古籍审美与学术规范的数字动态排版

- ✓ LaTeX 宏包深度定制，支持纵排、双行小注、版式边框
- ✓ Flutter 跨平台渲染，在移动端和 Web 端完美呈现古籍原貌
- ✓ 标准化模板库：宋版、明刻、清稿本等经典版式
- ✓ 古籍字体适配：集成思源宋体、开源古籍字体

**当前状态**: 🔨 **进行中**

[了解详情 →](/read/phase1)

---

### 第二阶段：完整校对工具集 ✨

**核心目标**：建立"图文对照"的高性能校对环境，确保数字化质量

- ✓ 图文联动对比器：扫描原件与 OCR 文本精准对应
- ✓ 同步滚动：点击文本定位图片，滚动图片高亮文本
- ✓ 异体字转换器：常用异体字、通假字快速替换
- ✓ 多人协作校对：基于 Git 的版本控制与差异比对

**当前状态**: 🔨 **进行中**

[了解详情 →](/read/phase2)

---

### 第三阶段：AI 辅助 OCR 提速 🔮

**核心目标**：大幅降低人工录入成本，提升数字化效率

- 集成开源领域先进的古籍 OCR 模型
- AI 智能断句、标点标注
- 自动文本结构提取
- 持续学习，不断提升识别准确率

**当前状态**: 📋 **规划中**

[了解详情 →](/read/phase3)

---

### 第四阶段：开源存储与资源共享 🔮

**核心目标**：建立开放的古籍数字图书馆，防止人类文化遗产流失

- 采用开放格式存储，支持版本管理
- 提供标准化 API 接口
- 参考维基文库、识典古籍等优秀项目
- 建立社区驱动的校对与审核机制

**当前状态**: 📋 **规划中**

[了解详情 →](/read/phase4)

---

### 第五阶段：知识图谱与 AI 模型训练 🔮

**核心目标**：从"数字化"走向"智能化"

- 构建人物、地点、事件、典籍关联的深度知识图谱
- 训练针对古汉语理解、翻译和研究的专用 AI 大模型
- 提供智能检索、语义分析、典故溯源等功能
- 推动古籍研究进入 AI 时代

**当前状态**: 📋 **规划中**

[了解详情 →](/read/phase5)

---

## 🛠️ 技术栈

- **前端框架**: Flutter (跨平台支持 Web / iOS / Android / Desktop)
- **排版引擎**: LaTeX 扩展宏包
- **后端存储**: 开源数据库方案
- **智能技术**: OCR 模型 & 大语言模型 (LLM)
