# 开源古籍 (Kaiyuan Guji)

> 让科技赋予古籍数字生命

本仓库包含开源古籍项目的官方网站。项目现已从 Flutter 迁移至 **Next.js (App Router)** 架构，以提供更好的性能、SEO 和开发体验。

通过技术手段推动古籍的数字化、校对及开源存储，构建古籍知识图谱与 AI 模型。

## 核心功能

- **古籍索引**：标准化的古籍资源导航，支持按类型筛选和实时搜索。
- **现代化阅读**：优化的宋体排版与移动端适配。
- **项目透明度**：实时同步的项目路线图与开发进展。

网站源代码位于 `nextjs/` 目录下。

```bash
cd nextjs
npm install
npm run dev
```
